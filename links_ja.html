論文調査・執筆・発表お役立ちサイト

  • Cryptology ePrint Archive
    暗号関連の論文がメインのアーカイブサイトです．
  • arXiv.org
    コンピュータサイエンス，数学，物理など幅広い分野の論文のアーカイブサイトです．
  • Weblio英和辞典・和英辞典
    論文を読むときに，この Web 辞書サイトをよく使います．
  • Mac.Tex
    理系の論文の多くは Tex とよばれる組版処理システムで書かれているはず．数式がとても綺麗に表示できます．私は Mac ユーザなので MacTex をインストールしています．難点はインストールに必要なパソコンの容量がかなり大きいここと，ダウンロードが時間がかかることです．Word で書いた文章より Tex で書いた文章の方が読みやすい．
  • LaTeXコマンドシート一覧
    Tex で数式を書いているときに，数学記号を出すためのコマンドをよくこのサイトで調べて書いています．
  • Google翻訳
    このツールが便利であることは疑う余地はないです．Google 翻訳の言語を検出するで「日本語」を選択して，左側に英語を入力して左下の音声を聞くを押してみてください．きっと日本語英語（カタカナ英語）を発音してくれます．
  • GitHub
    論文を執筆すると様々な部分を付け加えて書いたり削ったり変更が生じます．そのため，論文のバージョン管理が大切です．論文の Tex ファイルを GitHub とよばれるソースコード管理サービスでバージョン管理すると便利です．
  • SourceTree
    Github では CUI （キャラクタユーザインタフェース）で管理しようとすると，Windows ではコマンドプロンプト，Mac ではターミナルを使うので，CUI に慣れていない人にとってハードルが高いと思われます． SourceTree を用いることで GUI キャラクタユーザインタフェース）でのファイル管理が可能になり，Github ファイル管理が視覚的に分かりやすく行うことができるので便利です．
  • Grammarly
    英文法やスペルミスなどの自動添削機能がかなり役に立ちます．
  • NaturalReader
    国際会議の発表は英語ですが英語の発音がいつも気になります．そこで，英語の発表原稿が完成したら NaturalReder に読んでもらって発音の練習をします．しかし，この練習法には欠点があって，棒読みのような英語になってしまうことです．イントネーションも発表では大事です．
